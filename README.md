## Run Locally

Clone the project

```bash
  git clone https://github.com/Docat0209/ai-assistant
```

Go to the project directory

```bash
  cd ai-assistant
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server(using -sync to sync splah command)

```bash
  python src/main.py
```
